# PRACTICA_01_SEVP

Repositorio de práctica de manejo de Git.

## Descripción

Página web simple con múltiples archivos HTML, hojas de estilo y scripts.
Incluye manejo de ramas, merges, resolución de conflictos y uso de `.gitignore`.

## Estructura del proyecto

```
PRACTICA_01_SEVP/
├── index.html
├── page_a.html
├── page_b.html
├── styles/
│   └── style.css
├── scripts/
│   └── script.js
├── .gitignore
└── README.md
```

## Ramas

- `main` — rama principal
- `feature_1` — navbar y tabla en index/page_a
- `feature_2` — estilos, tabla en page_b e imagen en index
- `fix_1` — corrección de resta en script.js
